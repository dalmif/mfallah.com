# mfallah.com

This is an Astro project for my personal tech blog at **mfallah.com**, where I post my ideas, opinions, and talks about engineering and development.

## Design

Keep the site simple and clean — a minimal, typical tech personal blog. No fancy visuals, animations, or flashy design.

## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.
